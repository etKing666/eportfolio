Object Oriented
Computer Science

This module focuses on the fundamental concepts of objectoriented programming.
Python programming language and Unified Modelling Language (UML) is used
throughout the module.

The learing objectives of the module are:
- To be able to appraise and critically evaluate object-oriented programming compared to other programming paradigms.,
- To design and implement programs that demonstrate appropriate use of object-oriented design principles,
- To be able to select and use appropriate data structures for a given problem,
- To be able to Propose object-oriented solutions using an appropriate modelling language, such as UML.
(Source: https://online.essex.ac.uk/wp-content/uploads/2022/03/OOP_PCOM7E_PG_COM.pdf Accessed: 18.06.2022)
