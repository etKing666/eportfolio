The artefacts I have created during the Module 2 (Object Oriented Programming) of MSc Computer Science program.

A library inventory app in Python

Overview:

A library application was designed and developed in Python as part of an assignment in Module 1.

The algortihms and the data structure was decided and designed before coding. As per the assignment requirements, no external libraries were used.


Figure 1: Library inventory app.

Key fetures of the app:

• Insertion of new books • Deletion of books • Searching for a book by its name and author • Querying the number of the books in the inventory • Printing the list of books to the screen • The app features a text-based interface

You can access the source code and design document in the my Github repository.

SQL programming exercise

Overview:

In this assignment, EMP and DEPT tables from a fictional organisation “COMPANY1” were given. The student was expected to provide the SQL scripts to provide the desired results in given scenarios/problems.

Desired outcomes:

• List all Employees whose salary is between 1,000 AND 2,000. Show the Employee Name, Department and Salary • Count the number of people in department 30 who receive a salary and the number of people who receive a commission. • SFind the name and salary of employees in Dallas. • List all departments that do not have any employees. • List the department number and average salary of each department.

Check out the SQL scripts and the README file in which the methodology followed was discussed here.
