Launching into
Computer Science

In this module, we will explore the fundamental computing theories
as well as the challenges and threats with the development and adoption
of the new technologies.

The learing objectives of the module are:
- To be able to identify and explain the architecture, structure and functionality of basic components of a computer system,
- To understand the core data structures and programming concepts including algorithm computability,
- To be able to critically evaluate the functionality of different types of software,
- To be able to critically appraise the emerging trends in the field.
(Source: https://online.essex.ac.uk/wp-content/uploads/One-page-module-guides/Computing/LCS.pdf Accessed: 12.03.2022)
