Dissertation (Units 9-30):

Overview:

Instructions: The dissertation thesis must remain an original contribution to the field of research. It is a document of 13,000 words. It is expected that some key aspects of the code would be in the dissertation whilst the full code can be provided as a GitHub link/in the appendix.

Dissertation:

My research has led to the development of the SEANCE Framework, a cyber security framework tailored for microbusinesses. Based on the assumption that the most dangerous threat is the closest threat, the framework follows an inside-out, layered approach. It comprises six layers:

  1. Self
  2. Employees
  3. Assets
  4. Network
  5. Customers
  6. Environment

You can read more about the SEANCE Framework here.

I also developed SEANCE Threat Modelling Tool to facilitate the implementation of the framework. The tool poses a few straightforward, non-technical questions to the users and provides a risk score, recommendations and a data flow diagram based on their responses. The tool was developed using the Django Framework.

You can visit SEANCE Threat Modelling Tool here.
